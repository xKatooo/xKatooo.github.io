Node-SMS

Envia y recibe sms con node

Este esta hecho en node.js, es capaz de enviar mensajes SMS y recibirlos atravez de un numero virtual este si tiene mongo como base de datos, recibe los mensajes en tiempo real (con WebSockets) y muestra una pantalla con la informacion de los mensajes aparte de los logs en la terminal, este lo hice con un setup muy simple para que cualquiera pueda instalarlo y usarlo!
